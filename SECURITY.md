# Security

Do not report private community data or user information in a public issue.

Before sharing a security or privacy concern, remove credentials, webhook URLs, access tokens, raw private messages, user identifiers, payment details, and internal-only links. Use GitHub private vulnerability reporting when it is enabled for this repository.

The skills in this repository provide workflows and drafting guidance. They do not authorize public posting, permission changes, financial actions, reward settlement, moderation enforcement, or access to private systems.
